 Groq Code Architect (GCA)Groq Code Architect — это высокопроизводительная веб-платформа для автоматизированной генерации программного обеспечения. Используя мощь LPU-инференса от Groq, сервис мгновенно превращает идеи в структурированные кодовые базы, готовые к загрузке в виде ZIP-архивов. Live Service: https://project-grog-jdr9d6he6-ivanm696s-projects.vercel.app/ Ключевые особенностиГенерация на основе ИИ: Мгновенное создание кода любой сложности — от простых компонентов React до сложных Full-stack приложений.
Автоматическая архивация: Интеллектуальная сборка всех сгенерированных файлов в один .zip архив для удобного скачивания.
Безлимитный доступ: В приложение интегрирован высокоскоростной безлимитный токен, исключающий необходимость в настройке собственных ключей API пользователем.
Промышленная скорость: Благодаря оптимизации Groq, генерация проекта занимает считанные секунды.
Framework Agnostic: Поддержка множества языков и фреймворков (JavaScript, TypeScript, Python, HTML/CSS и др.).

 Технический стекЯдро ИИ: Groq Cloud API (Llama 3 / Mixtral).
Интерфейс: Vite + TypeScript + Tailwind CSS (с возможной миграцией на Next.js для SSR).
Логика файлов: Кастомный движок для маппинга JSON-ответов ИИ в виртуальную файловую систему.
Архивация: JSZip для клиентской сборки проектов.
Дополнительно: ESLint для линтинга, PostCSS для стилей, Vercel для деплоя.

 Как это работаетЗапрос: Пользователь вводит описание задачи (например, "Создание компонента React для графика Bitcoin").
Проектирование: ИИ анализирует запрос и выстраивает архитектуру файлов.
Генерация: Система пишет код для каждого файла (например, BitcoinChart.tsx).
Упаковка: Одним кликом все файлы собираются в архив с сохранением иерархии папок.

 Установка и запускКлонируйте репозиторий:

git clone https://github.com/ivanm696/project-grog.git

Установите зависимости:

cd project-grog
npm install

Запустите в режиме разработки:

npm run dev

Приложение будет доступно по адресу http://localhost:5173.Для продакшн-билда:

npm run build
npm run preview
